ClutchDownloader
================

A simple script that will download the latest version of <a href=https://github.com/KJCracks/Clutch>Clutch</a>

<b>Requirements:</b>

- wget
- sed
- gawk
- grep
- curl

Dependencies above get installed automatically when you start the script!
